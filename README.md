# Using Django's Auth API

A simple example of how to use Django's built in authentication API, as well as a simple admin setup.

To get it running locally:
1. Create and activate a new virtual env
2. `git clone https://github.com/tdsprogramming/django-auth-example.git`
3. `cd` into the project
4. `pip install -r requirements.txt`
5. `python manage.py makemigrations`
6. `python manage.py migrate`
7. `python manage.py runserver`
